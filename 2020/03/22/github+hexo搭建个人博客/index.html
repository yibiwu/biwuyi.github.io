github+hexo搭建个人博客

利用hexo+github搭建个人博客

1、什么是hexo

一个提供快速，简单并且很有效率的一个提供博客框架的平台

2、安装hexo

  1. 先下载node.js：直接去node.js中下载 ：https://nodejs.org/zh-cn/。我的是选择长期支持版的。

  2. 到git中（安装hexo）：输入命令行$ npm install hexo-deployer-git –save，后面出现版本号就是成功了。

  3. 建立文件夹 在文件夹中使用git输入hexo init来初始化文件。初始化完成之后使用：hexo install来安装依赖，接着安装插件：$ npm install hexo-deployer-git –save

  4. 到github中，创建仓库，但是这个要注意，仓库名建议要和博客命名一样，创建好之后，复制地址。找到创建文件夹中的seting文件 ：)之后更改设置：

  5. 更改完成，最后利用hexo g -d来将我们的博客部署到github上面，最后我们打开我们创建的博客就行了，输入的时候注意网址问题，例如我的是这种格式：https://yibiwu.github.io/，效果图：
